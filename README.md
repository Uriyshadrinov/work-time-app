Work Time App

Микросервис для учёта рабочего времени сотрудников с системой ролей и аналитикой.



Описание

Приложение позволяет отслеживать рабочее время сотрудников, фиксировать начало и конец рабочего дня, рассчитывать отработанные часы, опоздания и заработок.

Реализовано разделение прав доступа:

Администратор
Сотрудник

Функционал
 Employee (Сотрудник)
 Начало рабочего дня
 
 Завершение рабочего дня
 
 Подсчёт отработанных часов

 Учёт опозданий
 
 Просмотр статистики

 Просмотр заработка



 
 Admin (Администратор)
 
 
 Управление сотрудниками (CRUD)
 
 Просмотр общей статистики
 
 Отметка пропусков

 Аналитика (часы / заработок / опоздания)
 
 Просмотр активности



 Бизнес-логика:

Время работы считается автоматически

Заработок = часы × ставка

Опоздание считается от 09:00

Поддержка статусов:
present
absent
